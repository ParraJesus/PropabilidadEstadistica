Tercer Filtro

¿En qué consiste?

Al ingresar un dataset en la entrada, se calculará los percentiles y requerimientos necesarios para realizar el box plot de los datos. En cada iteración se eliminarán todos los elementos considerados outliners. Al final, para cada iteración se mostrará:

  • Media
  • Mediana
  • Desviación estándar
  • Extremo superior
  • Extremo inferior
  • Lista de ítems removidos
  • lista de valores restantes
  • Diagrama de caja
